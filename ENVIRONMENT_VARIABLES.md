# Environment Variables

A list of every _[Environment Variables](https://en.wikipedia.org/wiki/Environment_variable)_ used in **FunnyCoffee**.

| Name | Description | Default | Type |
| ---- | ----------- | ------- | ---- |
| `TOKEN` | The Discord bot token | `null` | `str` |
| `LAVALINK_HOSTNAME` | The hostname of Lavalink server to connect to | `127.0.0.1` | `str` |
| `LAVALINK_PORT` | The port of Lavalink server to connect to | 2333 | `int` |
| `LAVALINK_PASSWORD` | The password for Lavalink server | `youshallnotpass` | `str` |
| `MONGO_SRV` | _[MongoDB](https://www.mongodb.com/)_ connection URI | `null` | `str` |
